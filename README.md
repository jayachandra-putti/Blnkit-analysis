# Blinkit analysis Dashboard
## project objective  
The objective of this project is to analyze Blinkit’s sales, customer satisfaction, and inventory distribution using key KPIs.
It aims to uncover performance trends across product attributes and outlet characteristics.
The insights will support data-driven strategies to optimize growth, efficiency, and customer experience.

## Dataset used
- <a href="https://github.com/jayachandra-putti/Blnkit-analysis/blob/main/BlinkIT%20Grocery%20Data.xlsx">Dataset</a>

## KPIs &❓ Key Questions
- Total Sales
- Average Sales
- Number of Items
- Average Rating

 Key Questions
- How does fat content impact total sales and other KPIs?
- Which item types generate the highest sales and ratings?
- How do outlets differ in sales performance when segmented by fat content?
- Does the age or type of outlet establishment influence total sales?
- What is the percentage of sales by outlet size?
- How are sales distributed across different outlet locations?
- How do all KPIs vary by outlet type?
